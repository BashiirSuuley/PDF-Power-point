# 📄 → 📊 PDF to PowerPoint — Google Gemini AI

Upload any PDF → Gemini AI reads it → Beautiful `.pptx` downloaded instantly.
100% FREE — uses Google Gemini 1.5 Flash API (free tier).

## Get your FREE API key
1. Go to https://aistudio.google.com/app/apikey
2. Click "Create API Key"
3. Copy and paste into the app sidebar

## Files
- `app.py` — main app
- `requirements.txt` — streamlit + requests only
- `packages.txt` — installs Node.js on Streamlit Cloud
- `README.md` — this file

## Deploy on Streamlit Cloud
1. Upload all 4 files to GitHub
2. Go to share.streamlit.io → New app → your repo → app.py → Deploy
